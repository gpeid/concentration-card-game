# Concentration / Match Card Game
Quick matching game for my son to help boost memory, focus, and problem solving.

## Built with
- React
- TypeScript
- Vite
- TailwindCSS
- some custom css

## How to play
- Simple 52-card with suits and ranks.
- Select matching ranks to collect pairs.
- Current card selection are baby blue.
- Hovering over cards turns cards Fuchsia. 
- Matching pairs are indicated by green.
